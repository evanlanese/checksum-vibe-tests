1. Set localStorage 'products' to: [{"id": "enter-1", "name": "Keyboard Product", "description": "First item", "createdAt": "2024-01-01T00:00:00.000Z"}, {"id": "enter-2", "name": "Mouse Product", "description": "Second item", "createdAt": "2024-01-02T00:00:00.000Z"}]
2. Refresh the page
3. Verify both "Keyboard Product" and "Mouse Product" appear in the product list
4. Click in the search input field
5. Type "Keyboard" in the search input field
6. Press the Enter key
7. Verify "Keyboard Product" appears in the filtered product list
8. Verify "Mouse Product" does NOT appear in the filtered product list
9. Clear localStorage 'products' key
